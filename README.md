# Clash Royale

A Minecraft plugin that recreates the core gameplay mechanics of **Clash Royale** on Paper servers. Players can battle in real time by deploying troops, casting spells, and destroying enemy towers in a competitive arena.

> **Disclaimer:** This project is an unofficial fan project inspired by Clash Royale. It is not affiliated with, endorsed by, or associated with Supercell.


This project is still in very early development, it may not work as expected.

## Features planned
* Clash Royale inspired gameplay
* Set custom or auto generate arenas
* Progression system for players and cards

## Requirements

* Paper 21.1.2
* Java 25

## Building

Clone the repo and build it using Gradle.

### Linux/macOS

```bash
./gradlew build
```

### Windows

```cmd
gradlew.bat build
```

The compiled plugin can be found in:

```
build/libs/
```
